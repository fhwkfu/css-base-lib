css-base-library v1.01

基础组件

  • grid 栅格化
  • button 按钮
  • input 输入框
  • checkbox 复选框
  • radio 单选框
  • 附加

她，轻巧、灵动…
压缩后仅仅9KB，涵盖 24列栅格化、按钮、输入框、复选框和单选框，采用SASS一键编译，可任意定制你想要的样子（只需改变SASS变量既可），更多技巧等你来解锁，快快来体验吧。

如何使用

安装依赖 cnpm install
编译 npm run sass

grid 栅格化

提供24栏布局 超小屏幕手机 xs < 768，小屏幕 平板 sm ≥ 768，中等屏幕 桌面显示器 md ≥ 992，大屏幕 桌面显示器 lg ≥ 1200，超大屏幕 桌面显示器 xl ≥ 1920

常规用法，左右栅格

sl-col-10
sl-col-14
              
  <div class="sl-container">
    <div class="sl-row">
      <div class="sl-col-10">
        <div style="background: #fafafa; height: 100px;">sl-col-10</div>
      </div>
      <div class="sl-col-14">
        <div style="background: #fafafa; height: 100px;">sl-col-14</div>
      </div>
    </div>
  </div>

button 按钮

提供2种交互效果：hover、active，

a标签按钮 
              
  <button class="sl-button">按钮</button>
  <input type="button" value="主要按钮" class="sl-button">
  <input type="submit" value="提交按钮" class="sl-button">
  <input type="reset" value="重置按钮" class="sl-button">  
  <button class="sl-button" disabled>禁用按钮</button>
  <button class="sl-button outline">边框按钮</button> 
  <a class="sl-button outline">a标签按钮</a>

input 输入框

文本框、密码框、数字框、文本域

              
  <input type="text" class="sl-input" placeholder="文本框">
  <input type="password" class="sl-input" placeholder="密码框">
  <input type="number" class="sl-input" placeholder="数字框">
  <textarea autocomplete="off" placeholder="请输入内容" class="sl-textarea" style="width: 400px; height: 80px;"></textarea>

checkbox 复选框

4种状态

  <label for="c1">选择框</label>

  <input type="checkbox" value="value2" class="sl-checkbox" name="check" id="c2" checked>
  <label for="c2">选中</label>
  
  <input type="checkbox" value="value3" class="sl-checkbox" name="check" id="c3" disabled>
  <label for="c3">不可用</label>
  
  <input type="checkbox" value="value4" class="sl-checkbox" name="check" id="c4" disabled checked>
  <label for="c4">不可用选中</label>
